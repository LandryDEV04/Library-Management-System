# Library-Management-System
Ce projet implémente une base de données relationnelle complète pour la gestion d'une bibliothèque. Il couvre la conception (entités, relations, normalisation 3NF), l'implémentation MySQL (5 tables : authors,members,book,borrowing_records,fines)et le reporting via des requêtes SQL avancées (JOINs, GROUP BY, HAVING).
